# InmersionAgentes
Este repositorio contine la implementacion de un agente IA, para resolver consultas provenientes de un prompt. El agente resuelve de donde traer la informacion que se solicita en el prompt, si la informacion tiene que ver con temas internos de la empresa consulta en el RAG, si la informacion tiene que ver con temas generales busca en la WEB. En el RAG se encuentra segmentada la informacion referida a Comisiones de cobradores de los meses: Marzo, Abril, Mayo y Junio del año 2026, para ello se utilizaron Reportes de Comisiones (PDFs) generados en un sistema externo, los cuales se adjuntan.
