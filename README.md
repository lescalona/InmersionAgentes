# InmersionAgentes
Inmersion Agentes: Este repositorio contine la implementacion de un agente IA, para resolver consultas provenientes de un prompt, usando como RAG documentos sobre Reporte de Comisiones de los meses: Marzo, Abril, Mayo y Junio (se adjuntan).
El agente resuelve de donde traer la informacion que se solicita en el prompt, si la informacion tiene que ver con temas internos de la empresa consulta en el RAG, si la informacion tiene que ver con temas generales busca en la WEB.
